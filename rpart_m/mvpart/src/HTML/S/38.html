   2 /*
   3 ** The main entry point for recursive partitioning routines.
   4 **
   5 ** Input variables:
   6 **      n       = # of observations
   7 **      nvarx   = # of columns in xmat
   8 **      ncat    = # categories for each var, 0 for continuous variables.
   9 **      method  = 1 - anova
  10 **                2 - multivariate regression
  11 **                3 - exponential survival
  12 **                4 - classification
  13 **                5 - multivariate distances
  14 **                6 - user defined callback
  15 **      maxpri  = max number of primary variables to retain (must be >0)
  16 **      parms   = extra parameters for the split function, e.g. poissoninit
  17 **      ymat    = matrix or vector of response variables
  18 **      xmat    = matrix of continuous variables
  19 **      missmat = matrix that indicates missing x's.  1=missing.
  20 **      cptable  = a pointer to the root of the complexity parameter table
  21 **      tree     = a pointer to the root of the tree
  22 **      error    = a pointer to an error message buffer
  23 **      xvals    = number of cross-validations to do
  24 **      xgrp     = indices for the cross-validations
  25 **      wt       = vector of case weights
  26 **      opt      = options, in the order of rpart.control()
  27 **      ny       = number of columns in the input y matrix
  28 **      cost     = vector of variable costs
  29 **
  30 ** Returned variables
  31 **      error    = text of the error message
  32 **      which    = final node for each observation
  33 **
  34 ** Return value: 0 if all was well, 1 for error
  35 **
  36 */

 194     if (xvals >1 && (*tree)->rightson !=0) 
 195     xval(xvals, cptable, x_grp, maxcat, error, parms);

 197     ** all done
